ร่วมออกเดินทางไปกับเรา

เราเชื่อว่า ปทุมธานี คือหนึ่งในจังหวัดที่ใกล้และมีเสน่ห์ไม่แพ้ที่ใด หากคุณกำลังมองหาสถานที่พักผ่อนที่ไม่ต้องเดินทางไกล แต่ได้ประสบการณ์ที่หลากหลาย ลองให้ปทุมธานีเป็นจุดหมายปลายทางถัดไปของคุณทีมงานของเราหวังว่าข้อมูลนี้จะเป็นแรงบันดาลใจให้คุณออกไปค้นหาความสุขในแบบที่คุณชอบ แล้วพบกันที่ปทุมธานี!



1. วัดดอนใหญ่

ไฮไลท์ของ วัดดอนใหญ่ ที่สวยงดงามสุดๆ เลย คงจะหนีไม่พ้น อุโบสถสีเงิน ตั้งอยู่อย่างเด่นสวยสง่าเลย ซึ่งเป็นที่เดียวในจังหวัดปทุมธานีอีกด้วย ซึ่งอุโบสถหลังนี้มีทั้งสีขาวและสีเงิน เป็นสีที่แสดงถึงควาบริสุทธิ์ ส่วนกลีบบัวที่อยู่รอบอุโบสถนั้น ก็จะเป็นสีทองครับ เลยเป็นที่มาของชื่อ อุโบสถเงินในปทุมทอง ชื่อดังนั่นเอง


ที่อยู่ : 7 คลอง 8 ตำบลลำลูกกา อำเภอลำลูกกา จังหวัดปทุมธานี
พิกัด : https://maps.app.goo.gl/2pFmfSNt3223MxMEA
เปิดให้เข้าชม : สามารถเข้าชมได้ตามเวลาเปิดปิดของสถานที่
โทร : -
เว็บไซต์ : https://web.facebook.com/วัดดอนใหญ่


2. ตลาดอิงน้ำสามโคก

มาถึงอีกสิ่งของที่เที่ยวปทุมธานี ที่ต้องห้ามพลาด นั่นก็คือ ตลาดน้ำ ครับ ซึ่งถ้านึกถึงชุมชนริมน้ำก็ต้องมาที่ ตลาดอิงน้ำสามโคก แห่งนี้เลยครับ ภายในจะมีตลาดและของกินมากมาย ทั้งอาหารไทยดั้งเดิม อาหารมอญพื้นบ้าน รวมไปถึงสินค้าทางการเกษตรจากชาวบ้าน เป็นต้น โดยบรรยากาศรอบๆ ก็จะมีความเรียบง่าย สบายๆ ในสไตล์วิถีชุมชน และยังไม่ไกลจากกรุงเทพฯ ด้วย


ที่อยู่ : ถนนปทุมธานี-เสนา ตำบลสามโคก อำเภอสามโคก จังหวัดปทุมธานี

3. พิพิธภัณฑ์วิทยาศาสตร์แห่งชาติ

แลนด์มาร์คดีไซน์เก๋ กับตึกสวยทรงตึกลูกเต๋า ที่ต้องไปเช็คอินของ ปทุมธานี นั่นก็คือ พิพิธภัณฑ์วิทยาศาสตร์แห่งชาติ แห่งนี้ครับ โดยที่นี่จะจัดแสดงทั้ง ประวัติของวิทยาศาสตร์และเทคโนโลยี พลังงาน วิทยาศาสตร์ในชีวิตประจำวัน ภูมิปัญญาไทย การกำเนิดและวิวัฒนาการของสิ่งมีชีวิต ฯลฯ เป็นอีกที่เที่ยวที่ให้ความรู้ ควบคู่ไปกับความสนุกสนานเลยครับ


ที่อยู่ : ถนนเลียบคลองห้า ตำบลคลองห้า อำเภอคลองหลวง จังหวัดปทุมธานี
พิกัด : https://maps.app.goo.gl/MLSQDnGFNJeNzkXP8
เปิดให้เข้าชม : 09.30-17.00 น. (ปิดทำการทุกวันจันทร์)
โทร : -

4. Wisdom Farm พิพิธภัณฑ์การเกษตร

พิพิธภัณฑ์การเกษตรเฉลิมพระเกียรติฯ หรือ Wisdom Farm แห่งนี้ มีพื้นที่กว่า 300 ไร่ เป็นพิพิธภัณฑ์ทั้งในอาคาร และ พิพิธภัณฑ์การเรียนรู้กลางแจ้ง มีการจัดแสดงเกี่ยวกับการเกษตร เศรษฐกิจพอเพียง ไร่นาและพืชผักต่างๆ ที่เราจะได้ลองไปสัมผัสและลงมือทำด้วยตนเองครับ อีกทั้งยังมีมุมสวยๆ ไว้แชะภาพด้วย คือมุมของสะพานที่ทอดยาวผ่านทุ่งนานั่นเอง บอกเลยว่าถ่ายรูปรัวๆ แน่นอนครับ


ที่อยู่ : ริมถนนพหลโยธิน (ตรงข้ามโรงพยาบาลการุญเวช) นวนคร จังหวัดปทุมธานี
พิกัด : https://maps.app.goo.gl/ovrJoRq2P7LA8WYe8
เปิดให้เข้าชม : 09.00-16.00 น. (ปิดวันจันทร์ และวันหยุดนักขัตฤกษ์)
โทร : 0-2529-2212
เว็บไซต์ : https://www.facebook.com/WisdomKingMuseum/


5. ดงตาลสามโคก

ถ้าพูดถึง ที่เที่ยวถ่ายรูปสวยๆ ของ ปทุมธานี ต้องที่นี่เลยครับ ดงตาลสามโคก เป็นความงามของต้นตาลที่ปลูกเรียงกันอย่างโดดเด่นกลางทุ่ง พร้อมวิวของพระอาทิตย์ตก เหมาะสำหรับถ่ายภาพสวยๆ ฟีลกลางทุ่ง เป็นอีกที่เที่ยวธรรมชาติ และสถานที่พักผ่อนใกล้กรุง ที่สามารถมาเดินออกกำลังกาย หรือ ปั่นจักรยานเล่นชิลๆ ได้เลยครับ


ที่อยู่ : ตำบลบางโพธิ์เหนือ อำเภอสามโคก จังหวัดปทุมธานี
